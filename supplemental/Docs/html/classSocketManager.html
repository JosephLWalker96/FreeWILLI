void SocketManager::RestartListener ( )
inline

(Re)starts the udp listener. It closes the existing socket connection and creates a new one. Additionally, it clears the buffer (dataBuffer) and the segment to be processed (dataSegment) as well as the vector containing the timestamps (dataTimes).
